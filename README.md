# AI/ML Internship Task 4

## Project Title
General Health Query Chatbot

## Objective
The objective of this task is to build a simple chatbot that can answer general health-related questions using a Large Language Model.

## Tools and Libraries
- Python
- Google Colab
- Hugging Face Transformers
- FLAN-T5
- SentencePiece

## Model Used
google/flan-t5-base

## Features
- Takes user health-related questions
- Generates short and simple answers
- Uses prompt engineering
- Includes a safety filter for harmful medical queries
- Recommends consulting a doctor when needed

## Example Questions
- What causes sore throat?
- Is paracetamol safe for children?
- How can I reduce stress?

## Conclusion
This chatbot provides basic and safe health-related responses. It is not a replacement for professional medical advice.
